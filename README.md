# OPS102 Bonus Lab – Hangman Game (CMD Script)

This repository contains my Hangman game script written for the OPS102 Bonus Lab.  
The script is written entirely in Windows CMD batch scripting.

## Features
- Random word selection from a predefined list
- Hidden word mask that updates as letters are guessed
- Input validation (letters A–Z only)
- Duplicate letter detection
- Wrong guess counter and limit
- Win/Lose detection
- Scoreboard with win rate calculation
- ANSI colour output for a user-friendly experience
